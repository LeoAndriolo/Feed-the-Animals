# Feed the Animals

Simple Unity 3D prototype where the player feeds animals.

## Gameplay
- Throw food at animals
- Each animal has a hunger bar
- Animals require multiple hits to be fully fed
- Gain score by feeding animals
- Lose lives if animals reach the player

## Tech
- Unity (C#)
- Basic UI (score, lives, hunger bars)
